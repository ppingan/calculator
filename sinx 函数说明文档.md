# sinx 函数说明文档

## 概述
`sinx` 函数用于使用泰勒级数近似计算给定角度的正弦值。

## 函数签名
```python
def sinx(x: float) -> float:
    """
    使用泰勒级数近似计算给定角度的正弦值。

    参数：
    x (float): 输入的角度值（以度为单位）。

    返回：
    float: 给定角度的正弦值的近似值。
    """
```

## 参数

- `x`: 输入的角度值（以度为单位）。

## 返回值

- `float`: 给定角度的正弦值的近似值。

## 示例用法

```python
result = sinx(30)
print(result)  # 输出: 0.4999999994
```
